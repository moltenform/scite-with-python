### Download (Windows)

* unless you already have Python 2.7 installed, install [Python 2.7](https://www.python.org/downloads/windows/), use the Windows x86 MSI installer

* download [scite_with_python_0_7_5_win32.zip](https://github.com/moltenform/scite-with-python/releases/download/v0.7.5/scite_with_python_0_7_5_win32.zip)

* unzip scite_with_python_0_7_5_win32.zip

* open SciTE.exe

* (optional: <a href="dloadwin_src.html">build from source</a>)
